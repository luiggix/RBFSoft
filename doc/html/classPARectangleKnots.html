Detailed Description

template<typename Tprec>
class PARectangleKnots< Tprec >

PARectangular domain (NO detailed documentation).

This class is aimed to generate knots inside of rectangular domains. The function constructKnots() must be reimplemented here to generate the knots on the rectangular domain.

bool PARectangleKnots< prec_t >::constructKnots (  )  [inline]

Construct all the knots of the domain.

The way the knots are generated is tricky: Taking the values of Nx and Ny an "invisible grid" is defined inside the rectangle, this grid will contains (Nx - 1) X (Ny - 1) "cells", for each cell a knot is generated and this knot is allowed to be in a random position inside its cell, the randomness depends on the ep parameter.
