Detailed Description

template<typename Tprec>
class RectangleKnots< Tprec >

Rectangular domain.

This class is aimed to generate knots inside of rectangular domains. The function constructKnots() must be reimplemented here to generate the knots on the rectangular domain.

template<typename prec_t>
bool RectangleKnots< prec_t >::constructKnots (  )  [inline]

Construct all the knots of the domain.

This function generate nodes in a uniform grid (knd = 0), randomly distributed (knd = 1) and random controlled ( knd = 2, 0 < epru < 0.5).

The random controlled is generated as follows: Taking the values of Nx and Ny an "invisible grid" is defined inside the rectangle, this grid will contains (Nx - 1) X (Ny - 1) "cells", for each cell a knot is generated and this knot is allowed to be in a random position inside its cell, the randomness depends on the ep parameter.
